# **App Name**: SunnyBlooms

## Core Features:

- Animated Yellow Flowers Display: Displays a dynamic and animated scene of various yellow flowers, offering a visually pleasing experience.
- Animation Style Tool: Utilizes generative AI as a tool to interpret user text descriptions (e.g., 'gentle breeze', 'lively dance') and adjust the animation patterns and subtle movements of the yellow flowers accordingly.
- Adjustable Flower Parameters: Allows users to customize the density and overall speed of the animated flowers through simple interface controls.
- Background Ambience Selection: Provides a minimalist control to select simple background colors or gradients that complement the yellow flowers.

## Style Guidelines:

- Primary color: A vibrant, joyful golden yellow (#EBCE47) to embody warmth and happiness.
- Background color: A very light, subtle hint of yellow (#FDFAF5), almost off-white, to ensure the flowers stand out gracefully.
- Accent color: A soft coral-peach (#EEAE9D), chosen as an analogous hue to add a gentle, contrasting touch without being overpowering.
- Headline and body font: 'PT Sans', a humanist sans-serif, chosen for its modern, friendly, and approachable feel, aligning with the serene and natural theme.
- Minimalist and clean vector icons, possibly incorporating subtle floral motifs or petal shapes for navigational elements and controls.
- An immersive, full-screen layout that prioritizes the animated visual experience, with controls gracefully appearing on hover or discreetly placed to avoid distraction.
- Smooth, continuous looping animations for the flowers, simulating natural movements like swaying, gentle drifting, and subtle changes in scale to create a serene and lifelike atmosphere.
